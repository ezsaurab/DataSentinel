# DataSentinel

Enterprise Data Quality & Duplicate Intelligence Platform

## Vision

DataSentinel helps organizations identify duplicate records, measure data quality, and generate actionable insights from datasets.

## Learning Journey

This project is being built incrementally while learning Java OOP, Collections, File Handling, JDBC, Spring Boot, and System Design.
